What's New in SAFS

SAFSWINRelease2012.08.07

Final Update: 2012.08.07

Since: 2010.09.16

Here we provide info as to what has been added or fixed since the last Release (2010).

It should be noted that:

    SAFS for Windows--including Android Support--was released on 2012.08.07!
    Robotium Remote Control for Android testing new release on 2012.07.31!
    SAFS for the Apple Mac and IOS testing was released on 2012.06.22!

As you can see, SAFS is undergoing a very rapid pace of development in many areas expanding support in:

  • Core SAFS Framework robustness and performance
  • JSAFS Object Model (All Domains)
  • AQA Test Complete (NET/WPF/Flex)
  • Apple UIAutomation (Mac/IBT/IOS Apps)
  • Robotium Remote Control (Android)
  • Google Android SDK Automation (DRD Apps)
  • Selenium (Web)
  • IBM Rational Functional Tester (Flex/Web/Java/Net/WPF)
