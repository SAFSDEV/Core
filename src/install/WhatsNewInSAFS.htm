What's New in SAFS

Updated: 2015.10.26

Since: 2012.08.07

--- All active "development" is now at SAFSDEV on GitHub ---

(Releases/Updates and other Binaries will be available on both GitHub and SourceForge)

Here we provide info as to what has been added or fixed since the previous SAFS Win Release 2012.08.07 document.

Frequently Updated:

    Downloadable SAFS Updates from SourceForge|GitHub.
    Downloadable SAFS WIN Release Candidates from SourceForge|GitHub.

    Downloadable SeleniumPlus Updates from SourceForge|GitHub.

Less so:

    Download (older) Robotium RemoteControl Updates or Release Candidates for Robotium 5.2.1.
    (Note: The latest SAFS WIN Release Candidate above supports Remote Control with Robotium 5.3.1)

As you can see below, SAFS is undergoing a very rapid pace of development in many areas expanding support in:
  • Core SAFS Framework robustness and performance
  • Selenium WebDriver/SeInterpreter Support and Integration
  • SeleniumPlus Eclipse Test Development Environment
  • NEW! AutoIt WIN support for native Windows interactions.
  • SmartBear Test Complete (NET/WPF/Flex/Java/HTML)
  • IBM Rational Functional Tester (Flex/Web/Java/Net/WPF)
  • JSAFS Object Model (for ALL Domains -- ALL tools)
  • SAFS/Robotium Remote Control (Android)
  • SAFS Engine for Android Automation (DRD Apps)
